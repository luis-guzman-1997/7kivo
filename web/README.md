# 7kivo - Sitio Web Corporativo

Sitio web oficial de 7kivo, empresa salvadoreña especializada en modernización de procesos mediante tecnología e inteligencia artificial.

## Características

- Diseño moderno y responsivo
- Páginas informativas sobre la empresa y servicios
- Términos y Condiciones
- Política de Privacidad
- Formulario de contacto
- Integración con Font Awesome Free
- Bootstrap 5 para diseño responsivo
- Angular 17

## Tecnologías Utilizadas

- Angular 17
- Bootstrap 5
- Font Awesome Free
- TypeScript
- HTML5/CSS3

## Instalación

1. Instalar dependencias:
```bash
npm install
```

2. Ejecutar el servidor de desarrollo:
```bash
npm start
```

3. Abrir en el navegador:
```
http://localhost:4200
```

## Estructura del Proyecto

- `/src/app/components/home` - Página principal
- `/src/app/components/about` - Sobre nosotros
- `/src/app/components/services` - Servicios
- `/src/app/components/contact` - Contacto
- `/src/app/components/terms` - Términos y Condiciones
- `/src/app/components/privacy` - Política de Privacidad

## Build para Producción

```bash
npm run build
```

Los archivos compilados estarán en la carpeta `dist/7kivo-web`.
